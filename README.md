# The Quiet Crossroads

A small, free, browser-based choose-your-own-adventure story created with Twine (Harlowe format).

## How to Play
1. Open the website link (GitHub Pages URL)
2. Read the text
3. Click the choices to navigate the story

No downloads or accounts are required.

## How to Edit the Story
1. Go to https://twinery.org
2. Choose **Import From File**
3. Select `index.html`
4. Edit passages as desired
5. Use **Publish to File**
6. Replace `index.html` in this repository with the new version
